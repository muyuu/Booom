About Booom

This is jQueyr plugin at Add saturated linework over original image.

How to use

jQuery and plugin file load


<script src="http://code.jquery.com/jquery-1.9.1.min.js"></script>
<script src="./js/booom.js"></script>

Add class="saturated-linework" at img.

$(function(){
  $(".booom").Booom();
});
  • 画像を入れたdivのクラス名なり何なりをセレクタに。
  • オプションで画像パスを指定可能
  • width と height を指定して、中心からどれくらいずらすかを指定
